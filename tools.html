SimplePBI



SimplePBI es una librería desarrollada en Python que facilita el uso de la API REST de Power Bi.
Un día, SimplePBI contendrá todas las categorías en la API (administración, conjuntos de datos, informes, etc.). No dude en consultar el documento para obtener una comprensión más profunda de una solicitud específica:
- https://docs.microsoft.com/en-us/rest/api/power-bi/
Estamos haciendo todo lo posible para que esta biblioteca sea útil para la comunidad. Este proyecto no es un trabajo remunerado para nosotros. Es un proyecto público comunitario. Tenga paciencia si envía un problema y no se soluciona de inmediato. Es una forma de expresar nuestra pasión por compartir conocimientos.

Data Quality Report



El informe de calidad de datos es una herramienta externa (External Tool) para agregar en Power Bi. Crea un nuevo archivo de Power Bi Desktop a partir del modelo cargado en el primero. El nuevo archivo contendrá:

  • Primer vistazo a la estructura de las tablas y errores/espacios en blanco.
  • Comprensión de las distribuciones de datos de texto/fecha y número de cada tabla.
  • Obtener información de columnas categóricas. Categoría más utilizada para cada tabla.
  • Comprobar cómo se construyen las relaciones.
